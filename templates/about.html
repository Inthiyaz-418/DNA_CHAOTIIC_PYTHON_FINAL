📘 About the Chaotic-DNA Image Encryption System 🧬🔐


✨ Introduction

The Chaotic-DNA Image Encryption Algorithm is an advanced cryptographic technique 🌐 that combines the power of chaotic mathematical systems and DNA sequence operations. It provides extremely strong security by mixing pixel values using chaos theory and encoding them in a DNA-like structure for complex transformation. This approach is far more secure than traditional encryption because the output appears completely random 🔒✨.


🌀 How It Works – Overview

The algorithm uses two major processes:

  • 🧮 Chaotic Permutation – Shuffles pixel positions using logistic map randomness.
  • 🧬 DNA XOR Encryption – Converts each byte into DNA nucleotides (A, C, G, T) and performs DNA-level XOR operations for powerful confusion & diffusion.

📦 Module-Wise Explanation

1️⃣ Password Processing 🔑

  • Password is converted to a seed using SHA-256 hashing.
  • Seed generates chaotic sequence for pixel shuffling.

2️⃣ Chaotic Logistic Map Generator 🌀

  • Generates random-like sequence in range (0,1).
  • Used to create strong unpredictable pixel permutation.

3️⃣ Pixel Permutation Module 🔀

  • Entire image is flattened into a 1-D array.
  • Pixels are rearranged according to chaotic indices.

4️⃣ DNA Encoding Module 🧬

  • Each pixel byte is split into 4 two-bit segments.
  • Mapped to nucleotides (A, C, G, T) for DNA-based operations.

5️⃣ DNA XOR Substitution Module ⚡

  • DNA sequences are XORed with chaotic DNA keystream.
  • Produces highly scrambled output image.

6️⃣ Final Encrypted Image Generation 🖼️🔐

  • Encrypted array reshaped back to original dimensions.
  • Stored as PNG file.
